# Logs and Temporary Files

System logs, cache, and temporary files.

## Contents
- `tmp/` - Temporary files and working directory
- `__pycache__/` - Python compiled cache files

## Purpose
This directory contains:
- Runtime logs
- Temporary processing files
- Cache data
- System-generated files

## Management
Files in this directory can be safely deleted to free up space. They are automatically regenerated as needed.
