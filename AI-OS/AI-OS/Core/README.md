# AI OS Core Components

Main operating system components and engine files.

## Files
- `ai-autonomous-os.html` - Main desktop interface
- `ai-kernel-core.js` - AI kernel engine
- `ai-app-store-core.js` - App store functionality

## Features
- Desktop environment management
- Window system (drag, resize, snap)
- AI kernel integration
- System event handling

## Usage
Open `ai-autonomous-os.html` in web browser to launch the main desktop.
