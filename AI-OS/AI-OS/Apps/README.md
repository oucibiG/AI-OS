# AI OS Applications

System applications and utilities.

## Applications
- `ai-system-settings.html` - GUI customization and system settings
- `ai-task-manager.html` - Process and system monitoring
- `ai-file-manager.html` - File browser and management
- `ai-system-monitor.html` - Real-time system dashboard
- `ai-app-store.html` - Application store interface

## Features
- GUI customization (themes, colors, fonts, resolution)
- Process management and monitoring
- File system operations
- Real-time system metrics
- Application distribution

## Integration
All applications integrate with the main desktop through the AI kernel system.
